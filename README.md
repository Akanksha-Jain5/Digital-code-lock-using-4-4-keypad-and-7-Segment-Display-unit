# Digital-code-lock-using-4x4-keypad-and-7-Segment-Display-unit
Digital code lock circuit is implemented using 4×4 keypad and 1 SSD unit.4×4 keypad and one SSD unit is interfaced with microprocessor 8085 using 8255 Programmable Peripheral Interface (PPI).	Port A and upper Port C of 8255 PPI is used as output port whereas lower port C is used as input port.Port C is used for interfacing of 4×4 keypad whereas Port A is used for interfacing of SSD unit.Keypad has 16 keys which contains numbers from 0 to 9, alphabets A,B,C,D and two symbols *(RESET) and # (ENTER).
